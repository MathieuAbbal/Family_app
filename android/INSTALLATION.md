# Installation de FamilyApp sur Android

## Etape 1 : Recevoir le fichier APK

Recupere le fichier `app-debug.apk` envoye par USB, Telegram, WhatsApp ou autre.

## Etape 2 : Autoriser les sources inconnues

Comme l'app ne vient pas du Google Play Store, il faut autoriser l'installation :

1. Va dans **Parametres** > **Securite** (ou **Parametres** > **Applications**)
2. Active **Sources inconnues** ou **Installer des applis inconnues**
3. Si Android demande pour quelle app autoriser (ex: Chrome, Fichiers), autorise l'app depuis laquelle tu ouvres l'APK

> Sur Android 8+ : le reglage est par application. Par exemple si tu ouvres l'APK depuis **Fichiers**, autorise uniquement **Fichiers** a installer des apps.

## Etape 3 : Installer l'APK

1. Ouvre le fichier `app-debug.apk` depuis ton gestionnaire de fichiers ou tes telechargements
2. Appuie sur **Installer**
3. Si Google Play Protect affiche un avertissement, appuie sur **Installer quand meme** (l'app n'est pas sur le Play Store mais elle est sure)
4. Attends la fin de l'installation
5. Appuie sur **Ouvrir**

## Etape 4 : Se connecter

1. L'app s'ouvre sur la page de connexion
2. Appuie sur **Se connecter avec Google**
3. Choisis ton compte Google familial
4. C'est pret !

## En cas de probleme

| Probleme | Solution |
|----------|----------|
| "Installation bloquee" | Active les sources inconnues (etape 2) |
| "Play Protect a bloque l'installation" | Appuie sur "Plus de details" puis "Installer quand meme" |
| "Fichier non compatible" | Verifie que ton telephone est sous Android 6.0 minimum |
| L'app ne se lance pas | Desinstalle puis reinstalle l'APK |
